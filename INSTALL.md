### Install

**Download** `forge.exe` or `forge` binary, then:

```bash
chmod +x forge       # Linux/Mac only
./forge --install    # Install
```

Restart terminal, then:

```bash
forge --version      # Verify
```

### Uninstall

```bash
forge --uninstall
```

### Troubleshooting

**"forge not found"** - Restart your terminal

**Linux/Mac PATH issue:**
```bash
echo 'export PATH="$HOME/.local/bin:$PATH"' >> ~/.bashrc
source ~/.bashrc
```
